Materiais de som e imagem

A EquipmentHub foi formada em de Maio de 2020, por Luis David, Eliseu Batista, Bruno Pereira e por Sérgio Gonçalves.

A empresa tem como objetivo facilitar distrubuição de materiais de som e imagem, sem que se tenha que comprar os mesmos, para isso foi criado um Sistema Distribuido com a implementação de uma base de dados, para garantir a persistência de dados, que permite aos utilizadores reservarem e alugarem os materiais.
